# 🤖 4-Bar Linkage Simulator (Vibe Coding)

This project is a dedicated engineering tool for designing and analyzing **4-bar linkage systems**, the most fundamental 1-DOF mechanisms in robotics. Developed through **Vibe Coding**, this program focuses on providing high-fidelity kinematic verification for professional mechanical design.

## 🛠 Key Engineering Features

This program covers comprehensive technical details required for actual robotic mechanism design:

*   **Mechanism Generation & Angle Conversion**: Accurate configuration of link lengths and real-time visualization of angular motion.
*   **Grashof Condition Check**: Automated classification of mechanism types (Crank-Rocker, Double-Crank, Double-Rocker).
*   **Dead Point Analysis**: Identification of critical points in **Double-Rocker** systems to prevent motion failure.
*   **Instantaneous Center (IC) Visualization**: Real-time plotting of instantaneous centers for advanced velocity and force analysis.
*   **Free Prompt Sharing**: The exact prompts used to build this logic via Gemini are shared to support the engineering community.

## 🧠 Engineering Note: Vibe Coding

This repository demonstrates the power of "Vibe Coding" in mechanical engineering. The engineer defines the mathematical principles (Grashof's law, IC calculation, etc.), and Gemini rapidly iterates the GUI logic and `matplotlib` data visualization. The result is a professional-grade design tool developed with extreme efficiency.
